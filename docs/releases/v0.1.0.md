# Friday v0.1.0

Friday v0.1.0 is the first public MVP release of the local-first workflow
engine.

## Highlights

- Project-local `.friday/` memory initialization and status checks.
- Deterministic local evidence preparation and opt-in evidence collection.
- Planning and review prompt builders that write inspectable Markdown outputs
  and print local privacy, route, and advisory cost summaries.
- Privacy classification, common secret detection, and safe route composition.
- Advisory model route previews without provider execution.
- Advisory provider/model cost estimates from estimated token counts.
- Provider-neutral model contracts and a deterministic mock provider for future
  execution work.

## Boundaries

- No AI provider calls.
- No API key loading or provider configuration.
- No npm registry publication in this release.
- No usage telemetry, budget enforcement, or real billing records.

## Validation

- `npm run check`
- `npm pack`
- Packed CLI installed and smoke-tested in a clean external repository.
